Telnet Proxy Server created to access Karate Pizza BBS, the unofficial support BBS for TriBBS.

check it out via telnet @ tribbs.net:9000 

Karate Pizza runs on vintage hardware:

- Intel Celeron 333a
- 256mb Ram
- OS/2 Warp 4.52
- TriBBS 11.6 (4 Nodes)
